1. Write $p(t)$ as a linear combination of the basis: $p(t) = a_1\cdot 1 + a_2\cdot 2t + a_3\cdot (-2 + 4t^2) + a_4\cdot (-12t + 8t^3)$.
2. Expand the right side: $a_1 + 2a_2 t - 2a_3 + 4a_3 t^2 - 12a_4 t + 8a_4 t^3$.
3. Group like terms: $(a_1 - 2a_3) + (2a_2 - 12a_4)t + 4a_3 t^2 + 8a_4 t^3$.
4. Set equal to $p(t)$ and match coefficients: $7 - 12t - 8t^2 + 12t^3 = (a_1 - 2a_3) + (2a_2 - 12a_4)t + 4a_3 t^2 + 8a_4 t^3$.
5. Write the system: $a_1 - 2a_3 = 7$, $2a_2 - 12a_4 = -12$, $4a_3 = -8$, $8a_4 = 12$.
6. Solve $4a_3 = -8$ to get $a_3 = -2$.
7. Solve $8a_4 = 12$ to get $a_4 = \frac{3}{2}$.
8. Substitute $a_3$ into $a_1 - 2a_3 = 7$: $a_1 - 2(-2) = 7 \implies a_1 + 4 = 7 \implies a_1 = 3$.
9. Substitute $a_4$ into $2a_2 - 12a_4 = -12$: $2a_2 - 12 \cdot \frac{3}{2} = -12 \implies 2a_2 - 18 = -12 \implies 2a_2 = 6 \implies a_2 = 3$.
10. The coordinate vector of $p$ relative to $\mathcal{B}$ is $\begin{bmatrix} 3 \\ 3 \\ -2 \\ \frac{3}{2} \end{bmatrix}$.
