1. The inner product is defined as $\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)$, and we are to project $p_2(t) = t^2$ onto the span of $p_0(t) = 1$ and $p_1(t) = t$.

2. Compute the values: $p_0(-3) = 1$, $p_0(-1) = 1$, $p_0(1) = 1$, $p_0(3) = 1$; $p_1(-3) = -3$, $p_1(-1) = -1$, $p_1(1) = 1$, $p_1(3) = 3$; $p_2(-3) = 9$, $p_2(-1) = 1$, $p_2(1) = 1$, $p_2(3) = 9$.

3. Compute inner products: $\langle p_0, p_0 \rangle = 1+1+1+1=4$; $\langle p_1, p_1 \rangle = 9+1+1+9=20$; $\langle p_0, p_1 \rangle = (-3)+(-1)+1+3=0$; $\langle p_2, p_0 \rangle = 9+1+1+9=20$; $\langle p_2, p_1 \rangle = 9\cdot(-3)+1\cdot(-1)+1\cdot1+9\cdot3 = -27-1+1+27=0$.

4. The projection formula is $\mathrm{proj}_W(p_2) = \frac{\langle p_2, p_0 \rangle}{\langle p_0, p_0 \rangle}p_0 + \frac{\langle p_2, p_1 \rangle}{\langle p_1, p_1 \rangle}p_1 = \frac{20}{4}p_0 + \frac{0}{20}p_1 = 5\cdot 1 + 0\cdot t = 5$.

5. The orthogonal projection of $p_2$ onto the subspace spanned by $p_0$ and $p_1$ is $5$.

6. To find $q$ orthogonal to $p_0$ and $p_1$, let $q(t) = a + bt + ct^2$ and require $\langle q, p_0 \rangle = 0$ and $\langle q, p_1 \rangle = 0$.

7. Compute $\langle q, p_0 \rangle = a\cdot 4 + b\cdot 0 + c\cdot 20 = 4a + 20c = 0$.

8. Compute $\langle q, p_1 \rangle = a\cdot 0 + b\cdot 20 + c\cdot 0 = 20b = 0 \implies b = 0$.

9. From $4a + 20c = 0$, $a = -5c$.

10. So $q(t) = -5c + 0\cdot t + c t^2 = c(t^2 - 5)$.

11. The values of $q$ at $-3, -1, 1, 3$ are: $q(-3) = c(9-5) = 4c$, $q(-1) = c(1-5) = -4c$, $q(1) = c(1-5) = -4c$, $q(3) = c(9-5) = 4c$.

12. To scale so that $(q(-3), q(-1), q(1), q(3)) = (1, -1, -1, 1)$, set $4c = 1 \implies c = \frac{1}{4}$.

13. Thus, $q(t) = \frac{1}{4}(t^2 - 5)$.

14. Therefore, the orthogonal basis is $\{1, t, \frac{1}{4}(t^2 - 5)\}$, and the answers are: (a) The orthogonal projection is $5$; (b) $q(t) = \frac{1}{4}(t^2 - 5)$, with values $(1, -1, -1, 1)$ at $(-3, -1, 1, 3)$.
