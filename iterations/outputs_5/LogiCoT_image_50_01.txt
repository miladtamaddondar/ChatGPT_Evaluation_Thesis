1. The inner product on \( \mathbb{P}_3 \) is defined as \( \langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3) \); this is a valid inner product because it is linear, symmetric, and positive definite on polynomials evaluated at four points.
2. To compute the orthogonal projection of \( p_2 \) onto the subspace spanned by \( p_0 \) and \( p_1 \), we use the formula \( \mathrm{proj}_W(p_2) = a_0 p_0 + a_1 p_1 \), where \( a_0 = \frac{\langle p_2, p_0 \rangle}{\langle p_0, p_0 \rangle} \) and \( a_1 = \frac{\langle p_2, p_1 \rangle}{\langle p_1, p_1 \rangle} \), since \( p_0 \) and \( p_1 \) are already orthogonal.
3. Compute \( \langle p_0, p_0 \rangle = 1^2 + 1^2 + 1^2 + 1^2 = 4 \); this is correct because \( p_0(t) = 1 \) for all \( t \).
4. Compute \( \langle p_1, p_1 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9 + 1 + 1 + 9 = 20 \); this is correct since \( p_1(t) = t \).
5. Compute \( \langle p_2, p_0 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9 + 1 + 1 + 9 = 20 \); this is correct since \( p_2(t) = t^2 \) and \( p_0(t) = 1 \).
6. Compute \( \langle p_2, p_1 \rangle = (-3)^2(-3) + (-1)^2(-1) + 1^2(1) + 3^2(3) = 9(-3) + 1(-1) + 1(1) + 9(3) = -27 - 1 + 1 + 27 = 0 \); this is correct by direct calculation.
7. The projection is \( \mathrm{proj}_W(p_2) = \frac{20}{4}p_0 + \frac{0}{20}p_1 = 5p_0 \); this is correct because \( a_1 = 0 \).
8. The orthogonal projection of \( p_2 \) onto the subspace spanned by \( p_0 \) and \( p_1 \) is \( 5 \).
9. To find a polynomial \( q \) orthogonal to \( p_0 \) and \( p_1 \), let \( q(t) = at^2 + bt + c \) and require \( \langle q, p_0 \rangle = 0 \) and \( \langle q, p_1 \rangle = 0 \).
10. Compute \( \langle q, p_0 \rangle = a((-3)^2 + (-1)^2 + 1^2 + 3^2) + b((-3) + (-1) + 1 + 3) + c(1 + 1 + 1 + 1) = a(20) + b(0) + c(4) \); since \( b \) term cancels by symmetry.
11. Set \( \langle q, p_0 \rangle = 0 \Rightarrow 20a + 4c = 0 \Rightarrow c = -5a \).
12. Compute \( \langle q, p_1 \rangle = a((-3)^2(-3) + (-1)^2(-1) + 1^2(1) + 3^2(3)) + b((-3)^2 + (-1)^2 + 1^2 + 3^2) + c((-3) + (-1) + 1 + 3) = a(0) + b(20) + c(0) \).
13. Set \( \langle q, p_1 \rangle = 0 \Rightarrow 20b = 0 \Rightarrow b = 0 \).
14. So \( q(t) = a t^2 - 5a \); this is correct since it is orthogonal to both \( p_0 \) and \( p_1 \).
15. To scale \( q \) so that its values at \( (-3, -1, 1, 3) \) are \( (1, -1, -1, 1) \), compute \( q(-3) = a(9) - 5a = 4a \), \( q(-1) = a(1) - 5a = -4a \), \( q(1) = a(1) - 5a = -4a \), \( q(3) = a(9) - 5a = 4a \).
16. Set \( 4a = 1 \Rightarrow a = \frac{1}{4} \).
17. The scaled polynomial is \( q(t) = \frac{1}{4} t^2 - \frac{5}{4} \).
18. The orthogonal basis is \( \{1, t, \frac{1}{4}t^2 - \frac{5}{4}\} \), and the answer to part (b) is \( q(t) = \frac{1}{4}t^2 - \frac{5}{4} \).
