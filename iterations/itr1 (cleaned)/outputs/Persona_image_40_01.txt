1. The standard basis for \(\mathbb{P}_2\) is \(\{1, t, t^2\}\).  
2. The given basis is \(\mathcal{B} = \{1 - 3t^2,\, 2 + t - 5t^2,\, 1 + 2t\}\).  
3. To find the change-of-coordinates matrix from \(\mathcal{B}\) to the standard basis, write each vector in \(\mathcal{B}\) as a coordinate vector relative to the standard basis: \(1 - 3t^2 = 1 \cdot 1 + 0 \cdot t + (-3) \cdot t^2\), so its coordinate vector is \(\begin{bmatrix}1 \\ 0 \\ -3\end{bmatrix}\); \(2 + t - 5t^2 = 2 \cdot 1 + 1 \cdot t + (-5) \cdot t^2\), so its coordinate vector is \(\begin{bmatrix}2 \\ 1 \\ -5\end{bmatrix}\); \(1 + 2t = 1 \cdot 1 + 2 \cdot t + 0 \cdot t^2\), so its coordinate vector is \(\begin{bmatrix}1 \\ 2 \\ 0\end{bmatrix}\).  
4. The change-of-coordinates matrix \(P\) from \(\mathcal{B}\) to the standard basis is formed by placing these coordinate vectors as columns: \(P = \begin{bmatrix}1 & 2 & 1 \\ 0 & 1 & 2 \\ -3 & -5 & 0\end{bmatrix}\).  
5. To write \(t^2\) as a linear combination of the polynomials in \(\mathcal{B}\), set \(t^2 = a(1 - 3t^2) + b(2 + t - 5t^2) + c(1 + 2t)\) and expand: \(t^2 = a(1) + a(-3t^2) + b(2) + b(t) + b(-5t^2) + c(1) + c(2t)\).  
6. Collect like terms: \(t^2 = (a + 2b + c) + (b + 2c)t + (-3a - 5b)t^2\).  
7. Equate coefficients with \(t^2 = 0 \cdot 1 + 0 \cdot t + 1 \cdot t^2\): \(a + 2b + c = 0\), \(b + 2c = 0\), \(-3a - 5b = 1\).  
8. Solve the system: From \(b + 2c = 0\), \(b = -2c\); substitute into \(a + 2b + c = 0\): \(a + 2(-2c) + c = 0 \implies a - 4c + c = 0 \implies a = 3c\); substitute \(a = 3c\) and \(b = -2c\) into \(-3a - 5b = 1\): \(-3(3c) - 5(-2c) = 1 \implies -9c + 10c = 1 \implies c = 1\); thus \(a = 3\), \(b = -2\), \(c = 1\).  
9. Therefore, \(t^2 = 3(1 - 3t^2) - 2(2 + t - 5t^2) + 1(1 + 2t)\) as a linear combination of the polynomials in \(\mathcal{B}\).
