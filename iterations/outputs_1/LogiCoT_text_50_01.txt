1. Define the inner product on $\mathbb{P}_3$ as $\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)$. 
2. Compute the values: $p_0(t) = 1$ gives $(1, 1, 1, 1)$; $p_1(t) = t$ gives $(-3, -1, 1, 3)$; $p_2(t) = t^2$ gives $(9, 1, 1, 9)$. 
3. Compute $\langle p_0, p_0 \rangle = 1+1+1+1 = 4$, $\langle p_1, p_1 \rangle = 9+1+1+9 = 20$, $\langle p_0, p_1 \rangle = 1\cdot(-3) + 1\cdot(-1) + 1\cdot1 + 1\cdot3 = 0$. 
4. Compute $\langle p_2, p_0 \rangle = 9+1+1+9 = 20$, $\langle p_2, p_1 \rangle = 9\cdot(-3) + 1\cdot(-1) + 1\cdot1 + 9\cdot3 = -27-1+1+27 = 0$. 
5. The projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$ is $\frac{\langle p_2, p_0 \rangle}{\langle p_0, p_0 \rangle}p_0 + \frac{\langle p_2, p_1 \rangle}{\langle p_1, p_1 \rangle}p_1 = \frac{20}{4}p_0 + 0\cdot p_1 = 5p_0$. 
6. Verify orthogonality: $\langle p_2 - 5p_0, p_0 \rangle = \langle p_2, p_0 \rangle - 5\langle p_0, p_0 \rangle = 20-20=0$ and $\langle p_2 - 5p_0, p_1 \rangle = 0-5\cdot0=0$, so $p_2 - 5p_0$ is orthogonal to both $p_0$ and $p_1$. 
7. Let $q = p_2 - 5p_0 = t^2 - 5$ and compute its values: $q(-3) = 9-5=4$, $q(-1)=1-5=-4$, $q(1)=1-5=-4$, $q(3)=9-5=4$, so the vector is $(4, -4, -4, 4)$. 
8. To scale $q$ so its value vector is $(1, -1, -1, 1)$, divide by $4$: $q = \frac{1}{4}(t^2-5)$. 
9. The orthogonal basis is $\{p_0, p_1, q\}$ with $q = \frac{1}{4}(t^2-5)$, and the projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$ is $5p_0$.
