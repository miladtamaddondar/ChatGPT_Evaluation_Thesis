1. To find the vector $\mathbf{x}$ in the standard basis, write it as a linear combination of the basis vectors using the coordinates: $\mathbf{x} = 3\begin{bmatrix}1\\1\\3\end{bmatrix} + 2\begin{bmatrix}2\\-1\\1\end{bmatrix}$.
2. Multiply the first basis vector by 3: $3\begin{bmatrix}1\\1\\3\end{bmatrix} = \begin{bmatrix}3\\3\\9\end{bmatrix}$.
3. Multiply the second basis vector by 2: $2\begin{bmatrix}2\\-1\\1\end{bmatrix} = \begin{bmatrix}4\\-2\\2\end{bmatrix}$.
4. Add the two resulting vectors: $\begin{bmatrix}3\\3\\9\end{bmatrix} + \begin{bmatrix}4\\-2\\2\end{bmatrix} = \begin{bmatrix}7\\1\\11\end{bmatrix}$.
5. The vector $\mathbf{x}$ in the standard basis is $\boxed{\begin{bmatrix}7\\1\\11\end{bmatrix}}$.
