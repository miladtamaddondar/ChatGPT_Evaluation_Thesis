1. To find the coordinate vector of \( p(t) = 1 + 4t + 7t^2 \) relative to the basis \( \mathcal{B} = \{1 + t^2,\, t + t^2,\, 1 + 2t + t^2\} \), we express \( p(t) \) as a linear combination: \( p(t) = a(1 + t^2) + b(t + t^2) + c(1 + 2t + t^2) \).  
2. Expand the right side: \( a(1 + t^2) + b(t + t^2) + c(1 + 2t + t^2) = a + at^2 + bt + bt^2 + c + 2ct + ct^2 \).  
3. Combine like terms: \( (a + c) + (b + 2c)t + (a + b + c)t^2 \).  
4. Set equal to \( p(t) \): \( 1 + 4t + 7t^2 = (a + c) + (b + 2c)t + (a + b + c)t^2 \).  
5. Equate coefficients: \( a + c = 1 \), \( b + 2c = 4 \), \( a + b + c = 7 \).  
6. Solve the first equation for \( a \): \( a = 1 - c \).  
7. Substitute \( a \) into the third equation: \( (1 - c) + b + c = 7 \implies 1 + b = 7 \implies b = 6 \).  
8. Substitute \( b = 6 \) into the second equation: \( 6 + 2c = 4 \implies 2c = -2 \implies c = -1 \).  
9. Substitute \( c = -1 \) into \( a = 1 - c \): \( a = 1 - (-1) = 2 \).  
10. The coordinate vector of \( p(t) \) relative to \( \mathcal{B} \) is \( \boxed{\begin{pmatrix}2 \\ 6 \\ -1\end{pmatrix}} \).
