1. The inner product on \(\mathbb{P}_3\) is defined as \(\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)\).  
2. The polynomials are \(p_0(t) = 1\), \(p_1(t) = t\), \(p_2(t) = t^2\).  
3. To project \(p_2\) onto the subspace spanned by \(p_0\) and \(p_1\), we seek \(a, b\) such that the projection is \(a p_0 + b p_1\).  
4. The projection formula is \(\text{proj}_{U}(v) = \sum_{i=1}^k \frac{\langle v, u_i \rangle}{\langle u_i, u_i \rangle} u_i\) for an orthogonal basis \(\{u_1, u_2\}\) of \(U\); here, \(p_0\) and \(p_1\) are not orthogonal, so we use the general formula: projection = \(a p_0 + b p_1\) where \(p_2 - (a p_0 + b p_1)\) is orthogonal to both \(p_0\) and \(p_1\).  
5. Set up the orthogonality equations: \(\langle p_2 - (a p_0 + b p_1), p_0 \rangle = 0\) and \(\langle p_2 - (a p_0 + b p_1), p_1 \rangle = 0\).  
6. Compute \(\langle p_0, p_0 \rangle = 1^2 + 1^2 + 1^2 + 1^2 = 4\), \(\langle p_1, p_0 \rangle = (-3) + (-1) + 1 + 3 = 0\), \(\langle p_1, p_1 \rangle = 9 + 1 + 1 + 9 = 20\).  
7. Compute \(\langle p_2, p_0 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9 + 1 + 1 + 9 = 20\), \(\langle p_2, p_1 \rangle = (-3)^2(-3) + (-1)^2(-1) + 1^2(1) + 3^2(3) = -27 - 1 + 1 + 27 = 0\).  
8. The system is: \(\langle p_2, p_0 \rangle - a \langle p_0, p_0 \rangle - b \langle p_1, p_0 \rangle = 0\) and \(\langle p_2, p_1 \rangle - a \langle p_0, p_1 \rangle - b \langle p_1, p_1 \rangle = 0\).  
9. Substitute values: \(20 - 4a - 0b = 0 \implies a = 5\), \(0 - 0a - 20b = 0 \implies b = 0\).  
10. The orthogonal projection of \(p_2\) onto \(\text{span}\{p_0, p_1\}\) is \(5p_0\).  
11. To find \(q\) orthogonal to \(p_0\) and \(p_1\), let \(q(t) = p_2(t) - \text{proj}_{\text{span}\{p_0, p_1\}} p_2 = t^2 - 5\).  
12. Verify orthogonality: \(\langle q, p_0 \rangle = \langle t^2 - 5, 1 \rangle = 20 - 20 = 0\), \(\langle q, p_1 \rangle = \langle t^2 - 5, t \rangle = 0 - 0 = 0\).  
13. The orthogonal basis is \(\{p_0, p_1, q\}\) where \(q(t) = t^2 - 5\).  
14. Evaluate \(q\) at \(-3, -1, 1, 3\): \(q(-3) = 9 - 5 = 4\), \(q(-1) = 1 - 5 = -4\), \(q(1) = 1 - 5 = -4\), \(q(3) = 9 - 5 = 4\).  
15. To scale \(q\) so that its values at \((-3, -1, 1, 3)\) are \((1, -1, -1, 1)\), set \(q_{\text{scaled}}(t) = \frac{1}{4}(t^2 - 5)\).  
16. Final answer: The orthogonal projection of \(p_2\) onto \(\text{span}\{p_0, p_1\}\) is \(5\), and the scaled orthogonal polynomial is \(q(t) = \frac{1}{4}(t^2 - 5)\).
