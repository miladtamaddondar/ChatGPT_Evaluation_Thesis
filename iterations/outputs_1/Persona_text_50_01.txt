1. The inner product on $\mathbb{P}_3$ is defined as $\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)$.  
2. To project $p_2$ onto the subspace spanned by $p_0$ and $p_1$, we use the formula $\text{proj}_W(p_2) = a_0 p_0 + a_1 p_1$, where $a_i = \frac{\langle p_2, p_i \rangle}{\langle p_i, p_i \rangle}$ for $i=0,1$.  
3. Compute $\langle p_2, p_0 \rangle = p_2(-3)p_0(-3) + p_2(-1)p_0(-1) + p_2(1)p_0(1) + p_2(3)p_0(3) = 9\cdot1 + 1\cdot1 + 1\cdot1 + 9\cdot1 = 20$.  
4. Compute $\langle p_0, p_0 \rangle = 1^2 + 1^2 + 1^2 + 1^2 = 4$.  
5. Compute $\langle p_2, p_1 \rangle = p_2(-3)p_1(-3) + p_2(-1)p_1(-1) + p_2(1)p_1(1) + p_2(3)p_1(3) = 9\cdot(-3) + 1\cdot(-1) + 1\cdot1 + 9\cdot3 = -27 - 1 + 1 + 27 = 0$.  
6. Compute $\langle p_1, p_1 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9 + 1 + 1 + 9 = 20$.  
7. Thus, $a_0 = \frac{20}{4} = 5$ and $a_1 = \frac{0}{20} = 0$.  
8. Therefore, the orthogonal projection is $\text{proj}_W(p_2) = 5p_0 + 0p_1 = 5$.  
9. To find a polynomial $q$ orthogonal to $p_0$ and $p_1$, let $q(t) = p_2(t) - \text{proj}_W(p_2) = t^2 - 5$.  
10. Check orthogonality: $\langle q, p_0 \rangle = \langle t^2 - 5, 1 \rangle = \langle t^2, 1 \rangle - 5\langle 1, 1 \rangle = 20 - 20 = 0$ and $\langle q, p_1 \rangle = \langle t^2 - 5, t \rangle = \langle t^2, t \rangle - 5\langle 1, t \rangle = 0 - 0 = 0$ (since $\langle t^2, t \rangle = 0$ and $\langle 1, t \rangle = 0$ by symmetry).  
11. The vector of values of $q$ at $(-3, -1, 1, 3)$ is $((-3)^2-5, (-1)^2-5, 1^2-5, 3^2-5) = (4, -4, -4, 4)$.  
12. To scale $q$ so its values are $(1, -1, -1, 1)$, set $q(t) = \frac{1}{4}(t^2 - 5)$.  
13. Therefore, the orthogonal basis is $\{p_0, p_1, q\}$ where $q(t) = \frac{1}{4}(t^2 - 5)$.
