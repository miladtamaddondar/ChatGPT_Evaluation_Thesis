1. The inner product on $\mathbb{P}_3$ is defined as $\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)$, which is a valid inner product because it is a sum of products at distinct points (linearity and positive-definiteness hold).
2. To find the orthogonal projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$, write $p_2^\perp = p_2 - \text{proj}_{W}(p_2)$, where $W = \text{Span}\{p_0, p_1\}$, and $\text{proj}_{W}(p_2) = a_0 p_0 + a_1 p_1$ with $a_0, a_1$ chosen so that $p_2 - (a_0 p_0 + a_1 p_1)$ is orthogonal to both $p_0$ and $p_1$.
3. Set up the orthogonality conditions: $\langle p_2 - (a_0 p_0 + a_1 p_1), p_0 \rangle = 0$ and $\langle p_2 - (a_0 p_0 + a_1 p_1), p_1 \rangle = 0$, which by linearity gives the system: $\langle p_2, p_0 \rangle = a_0 \langle p_0, p_0 \rangle + a_1 \langle p_1, p_0 \rangle$ and $\langle p_2, p_1 \rangle = a_0 \langle p_0, p_1 \rangle + a_1 \langle p_1, p_1 \rangle$.
4. Compute the necessary inner products: $p_0(t) = 1$, so $p_0(-3) = p_0(-1) = p_0(1) = p_0(3) = 1$; $p_1(t) = t$, so $p_1(-3) = -3$, $p_1(-1) = -1$, $p_1(1) = 1$, $p_1(3) = 3$; $p_2(t) = t^2$, so $p_2(-3) = 9$, $p_2(-1) = 1$, $p_2(1) = 1$, $p_2(3) = 9$.
5. Calculate $\langle p_0, p_0 \rangle = 1+1+1+1 = 4$, $\langle p_1, p_0 \rangle = (-3)\cdot1 + (-1)\cdot1 + 1\cdot1 + 3\cdot1 = 0$, $\langle p_1, p_1 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9+1+1+9 = 20$.
6. Calculate $\langle p_2, p_0 \rangle = 9+1+1+9 = 20$, $\langle p_2, p_1 \rangle = 9\cdot(-3) + 1\cdot(-1) + 1\cdot1 + 9\cdot3 = -27 - 1 + 1 + 27 = 0$.
7. The system for $a_0$ and $a_1$ is: $20 = 4a_0 + 0a_1$ and $0 = 0a_0 + 20a_1$, so $a_0 = 5$ and $a_1 = 0$.
8. Therefore, the orthogonal projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$ is $5p_0 + 0p_1 = 5$.
9. To verify, $p_2 - 5 = t^2 - 5$; check orthogonality: $\langle t^2 - 5, 1 \rangle = (9-5) + (1-5) + (1-5) + (9-5) = 4 -4 -4 +4 = 0$ and $\langle t^2 - 5, t \rangle = 9\cdot(-3) - 5\cdot(-3) + 1\cdot(-1) - 5\cdot(-1) + 1\cdot1 - 5\cdot1 + 9\cdot3 - 5\cdot3 = (-27+15) + (-1+5) + (1-5) + (27-15) = (-12) + 4 -4 +12 = 0$.
10. Thus, the orthogonal projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$ is $5$.
11. For part (b), let $q(t) = t^2 - 5$ (the component of $p_2$ orthogonal to $p_0$ and $p_1$), and scale $q$ so that its values at $(-3, -1, 1, 3)$ are $(1, -1, -1, 1)$.
12. Compute $q(-3) = (-3)^2 - 5 = 9 - 5 = 4$, $q(-1) = 1 - 5 = -4$, $q(1) = 1 - 5 = -4$, $q(3) = 9 - 5 = 4$; thus, $q$'s value vector is $(4, -4, -4, 4)$.
13. To scale to $(1, -1, -1, 1)$, divide $q$ by $4$, so the desired polynomial is $r(t) = \frac{1}{4}(t^2 - 5)$.
14. Therefore, the polynomial $q(t) = \frac{1}{4}(t^2 - 5)$ is orthogonal to $p_0$ and $p_1$, and $\{p_0, p_1, q\}$ is an orthogonal basis for $\text{Span}\{p_0, p_1, p_2\}$, with $q$ scaled as required.
