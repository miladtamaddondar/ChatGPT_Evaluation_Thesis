1. The inner product is defined as \(\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)\). 
2. Compute values: \(p_0(t) = 1\), \(p_1(t) = t\), \(p_2(t) = t^2\). 
3. Compute projections: \(\text{proj}_{W}(p_2) = \frac{\langle p_2, p_0 \rangle}{\langle p_0, p_0 \rangle}p_0 + \frac{\langle p_2, p_1 \rangle}{\langle p_1, p_1 \rangle}p_1\). 
4. Compute \(\langle p_2, p_0 \rangle = p_2(-3)\cdot1 + p_2(-1)\cdot1 + p_2(1)\cdot1 + p_2(3)\cdot1 = 9+1+1+9=20\). 
5. Compute \(\langle p_0, p_0 \rangle = 1+1+1+1=4\). 
6. Compute \(\langle p_2, p_1 \rangle = p_2(-3)\cdot(-3) + p_2(-1)\cdot(-1) + p_2(1)\cdot1 + p_2(3)\cdot3 = 9\cdot(-3) + 1\cdot(-1) + 1\cdot1 + 9\cdot3 = -27-1+1+27=0\). 
7. Compute \(\langle p_1, p_1 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9+1+1+9=20\). 
8. So, \(\text{proj}_{W}(p_2) = \frac{20}{4}p_0 + 0\cdot p_1 = 5p_0\). 
9. The orthogonal projection of \(p_2\) onto \(\text{span}\{p_0, p_1\}\) is \(5\). 
10. For part (b), let \(q(t) = at^2 + bt + c\), require \(\langle q, p_0 \rangle = 0\) and \(\langle q, p_1 \rangle = 0\). 
11. Compute \(\langle q, p_0 \rangle = a\langle t^2, 1 \rangle + b\langle t, 1 \rangle + c\langle 1, 1 \rangle = a\cdot20 + b\cdot0 + c\cdot4 = 20a + 4c = 0\). 
12. Compute \(\langle q, p_1 \rangle = a\langle t^2, t \rangle + b\langle t, t \rangle + c\langle 1, t \rangle = a\cdot0 + b\cdot20 + c\cdot0 = 20b = 0\). 
13. Thus, \(b=0\), \(20a+4c=0\) so \(c=-5a\), so \(q(t) = a(t^2-5)\). 
14. To scale \(q\) so that its values at \((-3,-1,1,3)\) are \((1,-1,-1,1)\), compute \(q(-3)=a(9-5)=4a\), \(q(-1)=a(1-5)=-4a\), \(q(1)=a(1-5)=-4a\), \(q(3)=a(9-5)=4a\). 
15. Set \(4a=1\), so \(a=\frac{1}{4}\), thus \(q(t)=\frac{1}{4}(t^2-5)\). 
16. The orthogonal polynomial is \(q(t) = \frac{1}{4}(t^2-5)\) and the orthogonal basis is \(\{1, t, \frac{1}{4}(t^2-5)\}\).
