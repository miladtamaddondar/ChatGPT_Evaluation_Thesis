1. To find the coordinate vector of $p(t)$ relative to $\mathcal{B}$, write $p(t) = c_1(1+t^2) + c_2(t+t^2) + c_3(1+2t+t^2)$ and expand both sides.  
2. Expanding the right side gives $c_1(1+t^2) + c_2(t+t^2) + c_3(1+2t+t^2) = (c_1 + c_3) + (c_2 + 2c_3)t + (c_1 + c_2 + c_3)t^2$.  
3. Set this equal to $p(t) = 1 + 4t + 7t^2$ to get the system: $c_1 + c_3 = 1$, $c_2 + 2c_3 = 4$, $c_1 + c_2 + c_3 = 7$.  
4. Solve $c_1 + c_3 = 1$ for $c_1$: $c_1 = 1 - c_3$.  
5. Substitute $c_1 = 1 - c_3$ into $c_1 + c_2 + c_3 = 7$ to get $(1 - c_3) + c_2 + c_3 = 7$, so $1 + c_2 = 7$, thus $c_2 = 6$.  
6. Substitute $c_2 = 6$ into $c_2 + 2c_3 = 4$ to get $6 + 2c_3 = 4$, so $2c_3 = -2$, thus $c_3 = -1$.  
7. Substitute $c_3 = -1$ into $c_1 = 1 - c_3$ to get $c_1 = 1 - (-1) = 2$.  
8. The coordinate vector of $p(t)$ relative to $\mathcal{B}$ is $\begin{bmatrix} 2 \\ 6 \\ -1 \end{bmatrix}$.
