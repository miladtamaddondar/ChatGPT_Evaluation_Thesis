1. The inner product on $\mathbb{P}_3$ is defined as $\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)$.
2. To compute the orthogonal projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$, recall that the projection is $\text{proj}_W(p_2) = a_0 p_0 + a_1 p_1$ where $a_0 = \frac{\langle p_2, p_0 \rangle}{\langle p_0, p_0 \rangle}$ and $a_1 = \frac{\langle p_2, p_1 \rangle}{\langle p_1, p_1 \rangle}$.
3. Compute $p_0(-3) = 1$, $p_0(-1) = 1$, $p_0(1) = 1$, $p_0(3) = 1$ so $\langle p_0, p_0 \rangle = 1^2 + 1^2 + 1^2 + 1^2 = 4$.
4. Compute $p_1(-3) = -3$, $p_1(-1) = -1$, $p_1(1) = 1$, $p_1(3) = 3$ so $\langle p_1, p_1 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9 + 1 + 1 + 9 = 20$.
5. Compute $p_2(-3) = 9$, $p_2(-1) = 1$, $p_2(1) = 1$, $p_2(3) = 9$ so $\langle p_2, p_0 \rangle = 9\cdot1 + 1\cdot1 + 1\cdot1 + 9\cdot1 = 9 + 1 + 1 + 9 = 20$.
6. Compute $\langle p_2, p_1 \rangle = p_2(-3)p_1(-3) + p_2(-1)p_1(-1) + p_2(1)p_1(1) + p_2(3)p_1(3) = 9\cdot(-3) + 1\cdot(-1) + 1\cdot1 + 9\cdot3 = -27 - 1 + 1 + 27 = 0$.
7. Thus, $a_0 = \frac{20}{4} = 5$ and $a_1 = \frac{0}{20} = 0$.
8. Therefore, the orthogonal projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$ is $5p_0 + 0p_1 = 5$.
9. To find a polynomial $q$ orthogonal to $p_0$ and $p_1$, let $q(t) = p_2(t) - \text{proj}_W(p_2) = t^2 - 5$.
10. Check orthogonality: $\langle q, p_0 \rangle = \langle t^2 - 5, 1 \rangle = \langle t^2, 1 \rangle - 5\langle 1, 1 \rangle = 20 - 20 = 0$ and $\langle q, p_1 \rangle = \langle t^2, t \rangle - 5\langle 1, t \rangle = 0 - 0 = 0$.
11. Evaluate $q$ at $(-3, -1, 1, 3)$: $q(-3) = 9 - 5 = 4$, $q(-1) = 1 - 5 = -4$, $q(1) = 1 - 5 = -4$, $q(3) = 9 - 5 = 4$ so the vector is $(4, -4, -4, 4)$.
12. To scale $q$ so its vector is $(1, -1, -1, 1)$, divide by $4$: $q(t) = \frac{1}{4}(t^2 - 5)$.
13. Final answers: (a) The orthogonal projection of $p_2$ onto $\text{Span}\{p_0, p_1\}$ is $5$; (b) The required polynomial is $q(t) = \frac{1}{4}(t^2 - 5)$, so $\{p_0, p_1, q\}$ is an orthogonal basis.
