1. Write the equation $T(\mathbf{x}) = (-1, 4, 9)$ as $(x_1 - 2x_2, -x_1 + 3x_2, 3x_1 - 2x_2) = (-1, 4, 9)$.  
2. Equate components to form a system: $x_1 - 2x_2 = -1$, $-x_1 + 3x_2 = 4$, $3x_1 - 2x_2 = 9$.  
3. Verify the system is consistent by checking if the third equation is a linear combination of the first two: Multiply the first equation by 3: $3x_1 - 6x_2 = -3$; add to the second equation: $(-x_1 + 3x_2) + (3x_1 - 6x_2) = 4 + (-3)$ gives $2x_1 - 3x_2 = 1$.  
4. Compare $2x_1 - 3x_2 = 1$ with the third equation $3x_1 - 2x_2 = 9$ to form a $2 \times 2$ system: $x_1 - 2x_2 = -1$, $-x_1 + 3x_2 = 4$.  
5. Solve the first equation for $x_1$: $x_1 = -1 + 2x_2$.  
6. Substitute $x_1$ into the second equation: $-(-1 + 2x_2) + 3x_2 = 4$ gives $1 - 2x_2 + 3x_2 = 4$, so $1 + x_2 = 4$.  
7. Solve for $x_2$: $x_2 = 3$.  
8. Substitute $x_2 = 3$ into $x_1 = -1 + 2x_2$: $x_1 = -1 + 2 \times 3 = 5$.  
9. Verify the solution in the third equation: $3x_1 - 2x_2 = 3 \times 5 - 2 \times 3 = 15 - 6 = 9$, which matches the right side.  
10. The solution is $\mathbf{x} = (5, 3)$.
