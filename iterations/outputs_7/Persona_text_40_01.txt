1. To find the change-of-coordinates matrix from $\mathcal{B}$ to the standard basis $\mathcal{E} = \{1, t, t^2\}$, write each vector in $\mathcal{B}$ as a coordinate vector in $\mathcal{E}$: $1 - 3t^2 = 1\cdot 1 + 0\cdot t + (-3)\cdot t^2$ gives $\begin{bmatrix}1\\0\\-3\end{bmatrix}$.
2. Similarly, $2 + t - 5t^2 = 2\cdot 1 + 1\cdot t + (-5)\cdot t^2$ gives $\begin{bmatrix}2\\1\\-5\end{bmatrix}$.
3. Also, $1 + 2t = 1\cdot 1 + 2\cdot t + 0\cdot t^2$ gives $\begin{bmatrix}1\\2\\0\end{bmatrix}$.
4. The change-of-coordinates matrix from $\mathcal{B}$ to $\mathcal{E}$ is $P_{\mathcal{E} \leftarrow \mathcal{B}} = \begin{bmatrix}1 & 2 & 1\\ 0 & 1 & 2\\ -3 & -5 & 0\end{bmatrix}$.
5. To write $t^2$ as a linear combination of $\mathcal{B}$, set $t^2 = a(1 - 3t^2) + b(2 + t - 5t^2) + c(1 + 2t)$ and expand.
6. Expanding gives $t^2 = (a + 2b + c) + (b + 2c)t + (-3a - 5b)t^2$.
7. Equate coefficients with $t^2 = 0\cdot 1 + 0\cdot t + 1\cdot t^2$ to get the system: $a + 2b + c = 0$, $b + 2c = 0$, $-3a - 5b = 1$.
8. From $b + 2c = 0$, get $b = -2c$.
9. Substitute $b = -2c$ into $a + 2b + c = 0$ to get $a + 2(-2c) + c = 0 \implies a - 4c + c = 0 \implies a = 3c$.
10. Substitute $a = 3c$ and $b = -2c$ into $-3a - 5b = 1$ to get $-3(3c) - 5(-2c) = 1 \implies -9c + 10c = 1 \implies c = 1$.
11. Then $b = -2c = -2$, $a = 3c = 3$.
12. Therefore, $t^2 = 3(1 - 3t^2) - 2(2 + t - 5t^2) + 1(1 + 2t)$.
