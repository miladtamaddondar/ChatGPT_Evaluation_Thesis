1. To find the coordinate vector of \( p(t) = 1 + 4t + 7t^2 \) relative to \( \mathcal{B} = \{1 + t^2, t + t^2, 1 + 2t + t^2\} \), write \( p(t) = a(1 + t^2) + b(t + t^2) + c(1 + 2t + t^2) \) for scalars \( a, b, c \).  
2. Expand the right side: \( a(1 + t^2) + b(t + t^2) + c(1 + 2t + t^2) = a + a t^2 + b t + b t^2 + c + 2c t + c t^2 \).  
3. Combine like terms: \( (a + c) + (b + 2c)t + (a + b + c)t^2 \).  
4. Set coefficients equal to those in \( p(t) \): \( a + c = 1 \), \( b + 2c = 4 \), \( a + b + c = 7 \).  
5. Solve for \( a, b, c \): From \( a + c = 1 \), \( a = 1 - c \).  
6. Substitute \( a \) into \( a + b + c = 7 \): \( (1 - c) + b + c = 7 \implies 1 + b = 7 \implies b = 6 \).  
7. Substitute \( b = 6 \) into \( b + 2c = 4 \): \( 6 + 2c = 4 \implies 2c = -2 \implies c = -1 \).  
8. Substitute \( c = -1 \) into \( a = 1 - c \): \( a = 1 - (-1) = 2 \).  
9. Verify: \( a + c = 2 + (-1) = 1 \), \( b + 2c = 6 + 2(-1) = 4 \), \( a + b + c = 2 + 6 + (-1) = 7 \), all correct.  
10. The coordinate vector of \( p(t) \) relative to \( \mathcal{B} \) is \( \begin{bmatrix} 2 \\ 6 \\ -1 \end{bmatrix} \).
