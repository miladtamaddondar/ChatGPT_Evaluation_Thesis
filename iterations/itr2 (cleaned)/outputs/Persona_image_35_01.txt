1. To find the coordinate vector of \( p(t) = 1 + 4t + 7t^2 \) relative to \( \mathcal{B} = \{1 + t^2, t + t^2, 1 + 2t + t^2\} \), we express \( p(t) \) as a linear combination: \( p(t) = a(1 + t^2) + b(t + t^2) + c(1 + 2t + t^2) \).  
2. Expand the right side: \( a(1 + t^2) + b(t + t^2) + c(1 + 2t + t^2) = a + a t^2 + b t + b t^2 + c + 2c t + c t^2 \).  
3. Combine like terms: \( (a + c) + (b + 2c)t + (a + b + c)t^2 \).  
4. Set equal to \( p(t) = 1 + 4t + 7t^2 \) and equate coefficients: \( a + c = 1 \), \( b + 2c = 4 \), \( a + b + c = 7 \).  
5. Solve the system: From \( a + c = 1 \), \( a = 1 - c \).  
6. Substitute \( a = 1 - c \) into \( a + b + c = 7 \): \( (1 - c) + b + c = 7 \implies 1 + b = 7 \implies b = 6 \).  
7. Substitute \( b = 6 \) into \( b + 2c = 4 \): \( 6 + 2c = 4 \implies 2c = -2 \implies c = -1 \).  
8. Substitute \( c = -1 \) into \( a = 1 - c \): \( a = 1 - (-1) = 2 \).  
9. The coordinate vector of \( p(t) \) relative to \( \mathcal{B} \) is \( \boxed{\begin{pmatrix} 2 \\ 6 \\ -1 \end{pmatrix}} \).
