1. Find the eigenvalues of \(A\) by solving \(\det(A - \lambda I) = 0\): \(\det\left(\begin{bmatrix}0-\lambda & 1\\-3 & 4-\lambda\end{bmatrix}\right) = (0-\lambda)(4-\lambda) - (-3)(1) = \lambda^2 - 4\lambda + 3 = 0\).  
2. Factor and solve the characteristic equation: \(\lambda^2 - 4\lambda + 3 = (\lambda-1)(\lambda-3) = 0\) so the eigenvalues are \(\lambda_1 = 1\) and \(\lambda_2 = 3\).  
3. Find the eigenvector for \(\lambda_1 = 1\): Solve \((A - I)\vec{v} = 0\) where \(A - I = \begin{bmatrix}-1 & 1\\-3 & 3\end{bmatrix}\), so the system is \(-v_1 + v_2 = 0\) and \(-3v_1 + 3v_2 = 0\), giving \(v_2 = v_1\), so one eigenvector is \(\vec{v}_1 = \begin{bmatrix}1\\1\end{bmatrix}\).  
4. Find the eigenvector for \(\lambda_2 = 3\): Solve \((A - 3I)\vec{v} = 0\) where \(A - 3I = \begin{bmatrix}-3 & 1\\-3 & 1\end{bmatrix}\), so the system is \(-3v_1 + v_2 = 0\) and \(-3v_1 + v_2 = 0\), giving \(v_2 = 3v_1\), so one eigenvector is \(\vec{v}_2 = \begin{bmatrix}1\\3\end{bmatrix}\).  
5. The basis \(\mathcal{B}\) that diagonalizes \(A\) is \(\left\{\begin{bmatrix}1\\1\end{bmatrix}, \begin{bmatrix}1\\3\end{bmatrix}\right\}\).
