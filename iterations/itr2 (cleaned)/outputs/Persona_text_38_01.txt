1. The basis $\mathcal{B}$ is $\{1, 2t, -2 + 4t^2, -12t + 8t^3\}$, and we want to write $p(t) = 7 - 12t - 8t^2 + 12t^3$ as a linear combination of these basis polynomials, i.e., $p(t) = a_1 \cdot 1 + a_2 \cdot 2t + a_3(-2 + 4t^2) + a_4(-12t + 8t^3)$.  
2. Expand the right-hand side: $p(t) = a_1 + 2a_2 t + a_3(-2) + a_3(4t^2) + a_4(-12t) + a_4(8t^3) = (a_1 - 2a_3) + (2a_2 - 12a_4)t + 4a_3 t^2 + 8a_4 t^3$.  
3. Match coefficients with $p(t) = 7 - 12t - 8t^2 + 12t^3$ to get the system: $a_1 - 2a_3 = 7$, $2a_2 - 12a_4 = -12$, $4a_3 = -8$, $8a_4 = 12$.  
4. Solve $4a_3 = -8$ to get $a_3 = -2$.  
5. Solve $8a_4 = 12$ to get $a_4 = \frac{12}{8} = \frac{3}{2}$.  
6. Substitute $a_3 = -2$ into $a_1 - 2a_3 = 7$ to get $a_1 - 2(-2) = 7 \implies a_1 + 4 = 7 \implies a_1 = 3$.  
7. Substitute $a_4 = \frac{3}{2}$ into $2a_2 - 12a_4 = -12$ to get $2a_2 - 12 \cdot \frac{3}{2} = -12 \implies 2a_2 - 18 = -12 \implies 2a_2 = 6 \implies a_2 = 3$.  
8. Therefore, the coordinate vector of $p$ relative to $\mathcal{B}$ is $\begin{bmatrix} 3 \\ 3 \\ -2 \\ \frac{3}{2} \end{bmatrix}$.
