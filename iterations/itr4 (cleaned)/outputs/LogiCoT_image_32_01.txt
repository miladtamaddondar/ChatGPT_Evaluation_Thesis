3. The first entry: \(3(1) + (-5)(3) + (-3)(-4) = 3 - 15 + 12 = 0\); this verifies the first component is zero.  
4. The second entry: \(6(1) + (-2)(3) + 0(-4) = 6 - 6 + 0 = 0\); this verifies the second component is zero.  
5. The third entry: \((-8)(1) + 4(3) + 1(-4) = -8 + 12 - 4 = 0\); this verifies the third component is zero.  
6. Since all components are zero, \(A\mathbf{w} = \mathbf{0}\), so \(\mathbf{w}\) is in \(\text{Nul }A\).
