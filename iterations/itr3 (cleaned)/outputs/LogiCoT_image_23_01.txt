1. Since \( T \) is linear, \( T(a\mathbf{u} + b\mathbf{v}) = aT(\mathbf{u}) + bT(\mathbf{v}) \) for any scalars \( a, b \). 
2. To find \( T(3\mathbf{u}) \), use linearity: \( T(3\mathbf{u}) = 3T(\mathbf{u}) = 3\begin{bmatrix}2\\1\end{bmatrix} = \begin{bmatrix}6\\3\end{bmatrix} \); this is correct by the property \( T(c\mathbf{x}) = cT(\mathbf{x}) \). 
3. To find \( T(2\mathbf{v}) \), use linearity: \( T(2\mathbf{v}) = 2T(\mathbf{v}) = 2\begin{bmatrix}-1\\3\end{bmatrix} = \begin{bmatrix}-2\\6\end{bmatrix} \); this is correct by the same property. 
4. To find \( T(3\mathbf{u} + 2\mathbf{v}) \), use linearity: \( T(3\mathbf{u} + 2\mathbf{v}) = 3T(\mathbf{u}) + 2T(\mathbf{v}) = 3\begin{bmatrix}2\\1\end{bmatrix} + 2\begin{bmatrix}-1\\3\end{bmatrix} = \begin{bmatrix}6\\3\end{bmatrix} + \begin{bmatrix}-2\\6\end{bmatrix} = \begin{bmatrix}4\\9\end{bmatrix} \); this is correct by the additivity and homogeneity properties of linear transformations.
