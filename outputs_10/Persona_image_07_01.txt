1. The matrix \([Qr_1 \ Qr_2 \ \ldots \ Qr_p]\) is formed by multiplying the matrix \(Q\) by each vector \(r_i\) and placing the results as columns. 2. Notice that if we form the matrix \(R = [r_1 \ r_2 \ \ldots \ r_p]\), then multiplying \(Q\) by \(R\) gives \(Q[r_1 \ r_2 \ \ldots \ r_p] = [Qr_1 \ Qr_2 \ \ldots \ Qr_p]\). 3. Therefore, \([Qr_1 \ Qr_2 \ \ldots \ Qr_p] = Q \cdot [r_1 \ r_2 \ \ldots \ r_p]\). 4. The final answer is: \([Qr_1 \ Qr_2 \ \ldots \ Qr_p] = Q R\), where \(R = [r_1 \ r_2 \ \ldots \ r_p]\).
