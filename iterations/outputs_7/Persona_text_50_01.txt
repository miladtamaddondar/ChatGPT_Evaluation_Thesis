1. The inner product is defined as $\langle f, g \rangle = f(-3)g(-3) + f(-1)g(-1) + f(1)g(1) + f(3)g(3)$.
2. To project $p_2$ onto $\text{Span}\{p_0, p_1\}$, write $\operatorname{proj}_W(p_2) = a p_0 + b p_1$ where $a, b$ are scalars.
3. Set up orthogonality: $p_2 - (a p_0 + b p_1)$ is orthogonal to both $p_0$ and $p_1$, so $\langle p_2 - (a p_0 + b p_1), p_0 \rangle = 0$ and $\langle p_2 - (a p_0 + b p_1), p_1 \rangle = 0$.
4. Compute $\langle p_2, p_0 \rangle = p_2(-3)\cdot1 + p_2(-1)\cdot1 + p_2(1)\cdot1 + p_2(3)\cdot1 = 9 + 1 + 1 + 9 = 20$.
5. Compute $\langle p_0, p_0 \rangle = 1^2 + 1^2 + 1^2 + 1^2 = 4$.
6. Compute $\langle p_1, p_0 \rangle = (-3)\cdot1 + (-1)\cdot1 + 1\cdot1 + 3\cdot1 = -3 - 1 + 1 + 3 = 0$.
7. Compute $\langle p_2, p_1 \rangle = p_2(-3)\cdot p_1(-3) + p_2(-1)\cdot p_1(-1) + p_2(1)\cdot p_1(1) + p_2(3)\cdot p_1(3) = 9\cdot(-3) + 1\cdot(-1) + 1\cdot1 + 9\cdot3 = -27 - 1 + 1 + 27 = 0$.
8. Compute $\langle p_1, p_1 \rangle = (-3)^2 + (-1)^2 + 1^2 + 3^2 = 9 + 1 + 1 + 9 = 20$.
9. The equations are $20 - 4a - 0b = 0$ and $0 - 0a - 20b = 0$, so $a = 5$ and $b = 0$.
10. Thus, the orthogonal projection is $\operatorname{proj}_W(p_2) = 5p_0 + 0p_1 = 5$.
11. To find $q$, subtract the projection from $p_2$: $q = p_2 - \operatorname{proj}_W(p_2) = t^2 - 5$.
12. The vector of values of $q$ at $(-3, -1, 1, 3)$ is $q(-3) = 9-5=4$, $q(-1)=1-5=-4$, $q(1)=1-5=-4$, $q(3)=9-5=4$, so the vector is $(4, -4, -4, 4)$.
13. To scale $q$ so its values are $(1, -1, -1, 1)$, divide by $4$: $\frac{1}{4}(t^2 - 5)$.
14. The orthogonal basis is $\{p_0, p_1, q\}$ where $q(t) = \frac{1}{4}(t^2 - 5)$.
