1. Let the columns of the matrix be \(\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3\), where \(\mathbf{a}_1 = \begin{bmatrix}1\\-1\\-1\\1\\1\end{bmatrix}\), \(\mathbf{a}_2 = \begin{bmatrix}2\\1\\4\\-4\\2\end{bmatrix}\), \(\mathbf{a}_3 = \begin{bmatrix}5\\-4\\-3\\7\\1\end{bmatrix}\); we will use the Gram-Schmidt process to construct an orthogonal basis.
2. Set \(\mathbf{u}_1 = \mathbf{a}_1 = \begin{bmatrix}1\\-1\\-1\\1\\1\end{bmatrix}\).
3. Compute the projection of \(\mathbf{a}_2\) onto \(\mathbf{u}_1\): \(\mathrm{proj}_{\mathbf{u}_1} \mathbf{a}_2 = \frac{\mathbf{a}_2 \cdot \mathbf{u}_1}{\mathbf{u}_1 \cdot \mathbf{u}_1} \mathbf{u}_1\); first, \(\mathbf{a}_2 \cdot \mathbf{u}_1 = 2\cdot1 + 1\cdot(-1) + 4\cdot(-1) + (-4)\cdot1 + 2\cdot1 = 2 - 1 - 4 - 4 + 2 = -5\), and \(\mathbf{u}_1 \cdot \mathbf{u}_1 = 1^2 + (-1)^2 + (-1)^2 + 1^2 + 1^2 = 1+1+1+1+1=5\).
4. So, \(\mathrm{proj}_{\mathbf{u}_1} \mathbf{a}_2 = \frac{-5}{5} \mathbf{u}_1 = -1 \cdot \mathbf{u}_1 = \begin{bmatrix}-1\\1\\1\\-1\\-1\end{bmatrix}\).
5. Compute \(\mathbf{u}_2 = \mathbf{a}_2 - \mathrm{proj}_{\mathbf{u}_1} \mathbf{a}_2 = \begin{bmatrix}2\\1\\4\\-4\\2\end{bmatrix} - \begin{bmatrix}-1\\1\\1\\-1\\-1\end{bmatrix} = \begin{bmatrix}3\\0\\3\\-3\\3\end{bmatrix}\).
6. Compute the projection of \(\mathbf{a}_3\) onto \(\mathbf{u}_1\): \(\mathbf{a}_3 \cdot \mathbf{u}_1 = 5\cdot1 + (-4)\cdot(-1) + (-3)\cdot(-1) + 7\cdot1 + 1\cdot1 = 5 + 4 + 3 + 7 + 1 = 20\), so \(\mathrm{proj}_{\mathbf{u}_1} \mathbf{a}_3 = \frac{20}{5} \mathbf{u}_1 = 4 \cdot \mathbf{u}_1 = \begin{bmatrix}4\\-4\\-4\\4\\4\end{bmatrix}\).
7. Compute the projection of \(\mathbf{a}_3\) onto \(\mathbf{u}_2\): \(\mathbf{a}_3 \cdot \mathbf{u}_2 = 5\cdot3 + (-4)\cdot0 + (-3)\cdot3 + 7\cdot(-3) + 1\cdot3 = 15 + 0 - 9 - 21 + 3 = -12\), and \(\mathbf{u}_2 \cdot \mathbf{u}_2 = 3^2 + 0^2 + 3^2 + (-3)^2 + 3^2 = 9+0+9+9+9=36\), so \(\mathrm{proj}_{\mathbf{u}_2} \mathbf{a}_3 = \frac{-12}{36} \mathbf{u}_2 = -\frac{1}{3} \mathbf{u}_2 = \begin{bmatrix}-1\\0\\-1\\1\\-1\end{bmatrix}\).
8. Compute \(\mathbf{u}_3 = \mathbf{a}_3 - \mathrm{proj}_{\mathbf{u}_1} \mathbf{a}_3 - \mathrm{proj}_{\mathbf{u}_2} \mathbf{a}_3 = \begin{bmatrix}5\\-4\\-3\\7\\1\end{bmatrix} - \begin{bmatrix}4\\-4\\-4\\4\\4\end{bmatrix} - \begin{bmatrix}-1\\0\\-1\\1\\-1\end{bmatrix} = \begin{bmatrix}5-4-(-1)\\-4-(-4)-0\\-3-(-4)-(-1)\\7-4-1\\1-4-(-1)\end{bmatrix} = \begin{bmatrix}2\\0\\2\\2\\-2\end{bmatrix}\).
9. Therefore, an orthogonal basis for the column space is \(\left\{ \begin{bmatrix}1\\-1\\-1\\1\\1\end{bmatrix}, \begin{bmatrix}3\\0\\3\\-3\\3\end{bmatrix}, \begin{bmatrix}2\\0\\2\\2\\-2\end{bmatrix} \right\}\).
